Introduction

The main point of entry in a web browser is the address bar. In the address bar, a user can enter the URL (Uniform Resource Locator) of an HTML document to retrieve and display. A URL consists of:
  1. A protocol scheme to use when retrieving the HTML document -- usually we use 'http://' or 'https://', but other schemes are also possible, such as 'file://' or 'data://'
  2. A host name, which is the name of the server on the internet from which to retrieve the HTML document.
  3. A path, which the actual location of the html document on the server.
