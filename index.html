Introduction

Welcome to Intro to Web Development. We are going to learn the basics of HTML (HyperText Markup Language), CSS (Cascading Style Sheets), and JavaScript.

Here are some things we are going to use throughout this course:

  • Github is a web site that stores source code -- we are going to use it to store all of the code we write during this course. You should already have a GitHub account you are using to access this web page. Our code repository on GitHub is located at IntroToWebDevelopment. You will have to sign in to GitHub to access this private repository, and I will add your account the allowed users list.
  • GitHub Codespaces provides an online, browser-based development environment for us to write and share code for this class. Just click our "literate space goggles" codespace link to connect to the codespace, or sign in to github.com and select the "codespaces" tab at the top of the page.
  • Codepen is a website that allows you to enter short snippets of HTML, CSS, and JavaScript, view how it renders, and save and share your work with others. You can't build a complex web page here, but you can test out short snippets, and we will use it a lot as a playground to learn new concepts. You should create an account on Codepen, since you can "fork" any Codepen example you are viewing, which creates a copy of it in your own account so you can play around with it and make changes that do not affect the original pen. I created a Intro to Web Development Collection for pens I create for this class.
  • Mozilla Developer Network (MDN) is hands-down the best site for reference and learnring material related to HTML, CSS, and Javascript. When you are learning any new HTML element, JavaScript property, or JavaScript function, it's worth your time to search for MDN's page on the subject. Don't waste your time on W3Schools, head to MDN to get the best and most useful information.
    • If you want information on an HTML element like the anchor a tag, search for "mdn html a" to try to get a search result from MDN.
    • For information on the CSS "border" property, search for "mdn css border"
    • For information on the JavaScript getElementById function, search for "mdn javascript getElementById"
  • If you need some placeholder text, go grab some at Lorem Ipsum
  • Placeholder images can be gotten from Placekitten or Lorem Picsum
  • Sample sounds can be found at Google Sound Library
